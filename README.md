## Quick Start



Clone the project

```bash
  git clone https://github.com/Zinminhtun-de/react-ts-bolierplate-v-1.0.0.git
```

Go to the project directory

```bash
  cd react-ts-bolierplate-v-1.0.0
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
